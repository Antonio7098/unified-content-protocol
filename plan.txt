 Phase 1: Python SDK Fixes

 1.1 Block Hashability

 File: /packages/ucp-python/src/ucp/block.py

 Add after line 214 (end of Block class):
 def __hash__(self) -> int:
     Hash based on block ID for use in sets and as dict keys.
     return hash(self.id)

 def __eq__(self, other: object) -> bool:
     Equality based on block ID.
     if not isinstance(other, Block):
         return NotImplemented
     return self.id == other.id
